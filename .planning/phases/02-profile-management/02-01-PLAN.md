## Objective

Create the profile data structure and state management hook that will power all profile editing functionality.

## Context

Phase 1 established the three-column layout with hardcoded profile data in App.tsx. We need to extract this into a proper data structure with state management before building the UI components.

## Tasks

<task id="1">
**Create Profile type definition**

Create `src/types/Profile.ts` with:
- Interface defining profile fields: photo (string | null), name (string), bio (string), age (number | null), location (string)
- Default profile constant with sensible initial values
- Export both for use throughout the app

Success: Type file exists with complete interface covering all 5 profile fields
</task>

<task id="2">
**Create useProfile hook**

Create `src/hooks/useProfile.ts` with:
- useState to hold profile data (initialized with default profile)
- Functions to update each field: updatePhoto, updateName, updateBio, updateAge, updateLocation
- Return object with profile data and update functions
- Use TypeScript for full type safety

Success: Hook provides complete state management for profile with typed update functions
</task>

<task id="3">
**Add photo upload utilities**

In `src/hooks/useProfile.ts`, add:
- handlePhotoUpload function that accepts File input
- FileReader to convert uploaded file to base64 data URL
- Update photo field with data URL for display
- Handle errors gracefully (invalid file types, size limits)

Success: Hook can handle photo file input and convert to displayable format
</task>
