## Objective

Build complete commenting system that allows visitors to post comments on published articles, view all comments in chronological order, and see commenter names with timestamps.

## Context

Phase 4 created ArticleDetail component that displays full article content. Now we need to add a comment section below the article with a form for posting and a list for viewing comments. Following Skyblog 2006 aesthetic with pink accents and dark theme.

## Tasks

<task id="1">
**Create Comment type and useComments hook**

Create `src/types/Comment.ts`:
- Comment interface with fields:
  - id: string (crypto.randomUUID())
  - articleId: string (which article this belongs to)
  - authorName: string
  - content: string (comment text)
  - createdAt: Date
- Export type

Create `src/hooks/useComments.ts`:
- State: comments array (Comment[])
- getCommentsForArticle(articleId): filters comments by articleId, sorted by createdAt ascending
- addComment(articleId, authorName, content): creates new comment with id and timestamp
- Returns: { getCommentsForArticle, addComment }
- Store in component state (backend integration in Phase 6)

Success: Comment type defined, hook manages comment CRUD operations
</task>

<task id="2">
**Create CommentForm component**

Create `src/components/Comments/CommentForm.tsx`:
- Accept onSubmit callback
- Form fields:
  - Name input (required, max 50 chars)
  - Comment textarea (required, max 500 chars)
  - Submit button: "Poster le commentaire"
- Validation: both fields required before submit
- Clear form after successful submit
- Style with Skyblog aesthetic:
  - Dark inputs (#333 background, white text)
  - Pink submit button (#FF1493)
  - Verdana font
  - Input borders (#555)
- Show character counts below inputs

Success: Comment form accepts name and text, validates, submits
</task>

<task id="3">
**Create CommentList component**

Create `src/components/Comments/CommentList.tsx`:
- Accept comments array
- Display each comment with:
  - Author name in bold
  - Timestamp formatted as DD/MM/YYYY à HH:MM
  - Comment content
  - Divider between comments
- Empty state: "Aucun commentaire pour le moment. Soyez le premier à commenter!"
- Style with Skyblog aesthetic:
  - Dark background per comment (#1a1a1a)
  - Author name in pink (#FF1493)
  - Timestamp in gray (#999)
  - Content in light gray (#CCC)
  - Border between comments (#444)

Success: Comments display in chronological order with metadata
</task>

<task id="4">
**Integrate comments into ArticleDetail**

In `src/components/Blog/ArticleDetail.tsx`:
- Import useComments hook, CommentForm, CommentList
- Get comments for current article: getCommentsForArticle(article.id)
- Add comments section below article content and tags
- Section header: "Commentaires ([count])"
- Render CommentList with comments
- Render CommentForm below list
- Wire form onSubmit to addComment
- Update comment count in header after submission

Success: Article detail shows comments section with form and list
</task>
