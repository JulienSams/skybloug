# Plan 01-01: Establish Skyblog Three-Column Layout

**Objective:** Create the foundational React application with pixel-perfect Skyblog three-column layout matching reference images.

## Context

This is the first phase of a greenfield React project recreating the authentic mid-2000s Skyblog aesthetic. The user has provided three reference images (`1200x680_sc_jony.webp`, `5316900.jpg`, `blog-1024x640.webp`) showing the classic three-column structure. User decisions from CONTEXT.md require:
- Exact pixel measurements from screenshots
- Original Skyblog fonts (Verdana/Arial era)
- Exact colors extracted via color picker
- Faithful recreation of borders, shadows, spacing

<task id="4" type="implementation" depends_on="3">
**Create SkyblogLayout component with three-column structure**

Create `src/components/Layout/SkyblogLayout.tsx` and `SkyblogLayout.css`:

```tsx
interface SkyblogLayoutProps {
  leftSidebar: React.ReactNode;
  content: React.ReactNode;
  rightSidebar: React.ReactNode;
}

export function SkyblogLayout({ leftSidebar, content, rightSidebar }: SkyblogLayoutProps) {
  return (
    <div className="skyblog-layout">
      <div className="skyblog-sidebar-left">{leftSidebar}</div>
      <div className="skyblog-content">{content}</div>
      <div className="skyblog-sidebar-right">{rightSidebar}</div>
    </div>
  );
}
```

Implement CSS using exact measurements from task 2:
- Use CSS Grid or Flexbox for layout
- Fixed widths for sidebars, flexible center
- Proper spacing/margins
- Ensure columns align correctly
</task>
