## Objective

Add image upload and YouTube video embedding capabilities to the TipTap editor, using toolbar buttons and base64 storage.

## Context

User wants toolbar button + file picker for images (like Phase 2 profile photo), and toolbar button + URL prompt for YouTube. Images stored as base64 data URLs in article content. No drag-and-drop for Phase 3.

## Tasks

<task id="1">
**Add image upload to toolbar**

In `src/components/Editor/EditorToolbar.tsx`:
- Add "Add Image" button to toolbar
- Button opens file picker (hidden input type="file" accept="image/*")
- On file selection:
  - Read file with FileReader (reuse pattern from Phase 2 useProfile)
  - Convert to base64 data URL
  - Insert into editor: editor.chain().focus().setImage({ src: dataUrl }).run()
- Add file size validation (max 5MB like Phase 2)
- Style button with Skyblog aesthetic

Success: Image upload button works, inserts base64 images into editor
</task>

<task id="2">
**Add YouTube embed to toolbar**

In `src/components/Editor/EditorToolbar.tsx`:
- Add "Add Video" button to toolbar
- Button shows prompt dialog asking for YouTube URL
- On URL submission:
  - Basic validation (contains 'youtube.com' or 'youtu.be')
  - Insert into editor: editor.chain().focus().setYoutubeVideo({ src: url }).run()
  - Show error message if invalid URL
- Style button with Skyblog aesthetic

Success: YouTube button prompts for URL and embeds video player
</task>

<task id="3">
**Configure TipTap Image extension**

In `src/components/Editor/ArticleEditor.tsx`:
- Ensure Image extension is configured to allow base64 data URLs
- Images should be inline in content, not floating
- Set reasonable default width (e.g., max 550px to fit center column)

Success: Images display correctly in editor with proper sizing
</task>

<task id="4">
**Configure TipTap YouTube extension**

In `src/components/Editor/ArticleEditor.tsx`:
- Ensure YouTube extension is properly initialized
- Set default dimensions (560x315 - standard YouTube embed size)
- Videos should display as embedded players in editor

Success: YouTube videos display as playable embeds in editor
</task>
